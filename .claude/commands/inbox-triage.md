# Inbox Triage

Process Gmail fast. Clear the noise, find what needs action.

## Instructions

1. Read Gmail unread emails. For each one, classify as:
   - Action needed — you must reply or do something
   - Worth reading — no action, but relevant
   - Ignore — promo, newsletter, irrelevant

2. Present a clean triage list:

### Needs action
- [Subject] — [what to do in one line]

### Worth reading
- [Subject] — [why it matters in one line]

### Ignore
- [count] promotional or irrelevant emails — skip all

3. For each action-needed email, ask:
   "Want me to draft a reply for any of these?"

4. If yes — draft it:
   - Use Serif's voice: casual, direct, clear, natural English
   - Short — get to the point fast
   - No corporate filler phrases
   - Save draft to Gmail if possible
   - Otherwise save to 00_Inbox/YYYY-MM-DD-email-draft-[slug].md

5. Also scan 00_Inbox/ for any unprocessed notes or captures
   that need attention.

## When to Use
When your inbox feels messy or overwhelming.
Run it once in the morning after /digest.
Type: /inbox-triage
