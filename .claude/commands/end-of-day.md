# End of Day

Close the day with intention. Capture wins, challenges, and tomorrow's focus.

## Instructions

1. **Read context.**
   - Read `40_Admin/goals.md` for this week's focus
   - Read `40_Admin/Tasks.md` for today's open tasks

2. **Ask one question.**
   Use AskUserQuestion: "Quick end-of-day check-in — how did today go?"
   Options: Great / Solid / Tough / Mixed

3. **Prompt for the three fills** one at a time using AskUserQuestion:
   a. "What was your biggest win today? (one line)"
   b. "What got in the way or stayed unfinished?"
   c. "What are your top 3 for tomorrow?"

4. **Update today's Daily Note in Notion.**
   Find today's Daily Log entry (Daily Log database: c7017459-3ee6-4037-8aaa-f7424cf77ffe).
   Update or append:
   - Wins today → from answer (a)
   - Challenges → from answer (b)
   - Tomorrow's top 3 → from answer (c)
   If no entry exists for today, create one with Title: [Weekday, Month Day] and Type: Note.

5. **Mark completed tasks.**
   Ask: "Did you finish any tasks from today's list?"
   If yes — note them. Tell the user to check them off in Tasks.md or Notion Tasks.

6. **Close with a one-line summary.**
   Format: "Day closed. [Win]. Tomorrow: [#1 priority]."

## When to Use
At the end of your day — after work, before bed, whenever you're done.
Type: /end-of-day
