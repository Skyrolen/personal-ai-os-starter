# Class Help

Academic support tuned to Serif's style and situation.

## Input
$ARGUMENTS

## Context to load first
- Read CLAUDE.md — Economics student, second language English, 
  summa cum laude target (GPA must stay above 3.89)
- Read 40_Admin/goals.md — check what's due and what's at stake

## Instructions

1. **Identify the task** from $ARGUMENTS or by asking:
   - Writing a paper or essay?
   - Discussion post or short response?
   - Summarizing a reading?
   - Preparing a presentation?
   - Understanding an economics concept?
   - Something else?

2. **For writing tasks:**
   - Use clear, natural English — not overly academic or stiff
   - Strong thesis, clean structure, no filler sentences
   - Short paragraphs, direct sentences
   - Ask first: "Do you have a draft to improve, or do you want me to write fresh?"
   - Always ask: "What's the word count or page requirement?"

3. **For concept questions:**
   - Explain simply first, add depth only if needed
   - Use real-world examples over textbook ones
   - Connect to things Serif already understands from economics

4. **For presentations:**
   - Build the outline first, then expand slide by slide
   - Separate what goes ON the slide vs what to SAY verbally
   - Keep slides minimal — ideas live in the speaking, not the deck

5. **For discussion posts:**
   - Clear position in the first sentence
   - 2-3 supporting points with real examples
   - End with a question or insight, not a summary

6. **Always write in Serif's voice:**
   - Direct, clear, natural
   - Simple English — not dumbed down, just clean
   - No fancy vocabulary, no academic filler phrases

7. **Save output** to 10_Projects/school/YYYY-MM-DD-[slug].md 
   if the folder exists, otherwise to 00_Inbox/

## When to Use
Any time you need help with school work.
Type: /class-help [what you need]
Example: /class-help I need to write a 500 word discussion post about market failures
